HTML Selectors

4 Types of Selectors:

  1. Universal selector:
    • Begin with a star
    • * {property : value; }
  2. Type selector:
    • Begin with element name
    • elementName {property : value; }
  3. Class selector:
    • Begin with a period
    • .className {property : value; }
    • Several elements can belong to one class.
    • One element can have multiple classes.
      • class = "class-1 class-2"
  4. ID selector:
    • Begin with a hashtag
    • #ID {property : value; }
    • An unique ID can only be used once on the same HTML page
    • Use case: having links redirect to a section on the current page

3 Ways to Combine Selectors

  1. The Grouping Selector: to group the same declarations of selector1 and selector2
    • selector1,
      • selector2 {
              color: white;
              background-color: black;
      }
  2. Chaining Selector
    • selector1, selector2 {
           color: red;
      }
    • color:red will apply to any elements that both have selector1 and selector2
    • This syntax works for any combo of selectors except for chaining more than 1 type selectors, because one element cannot have two type selectors at the same time.
