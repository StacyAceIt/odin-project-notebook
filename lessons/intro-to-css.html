Intro to CSS

4 Types of Selectors:

  1. Universal selector:
    • Begin with a star
    • * {property : value; }
  2. Type selector:
    • Begin with element name
    • elementName {property : value; }
  3. Class selector:
    • Begin with a period
    • .className {property : value; }
    • Several elements can belong to one class.
    • One element can have multiple classes.
      • class = "class-1 class-2"
  4. ID selector:
    • Begin with a hashtag
    • #ID {property : value; }
    • An unique ID can only be used once on the same HTML page
    • Use case: having links redirect to a section on the current page

3 Types of Combined Selectors

  1. The Grouping Selector: to group the same declarations of selector1 and selector2
    • selector1,
      • selector2 {
              color: white;
              background-color: black;
      }
  2. Chaining Selector
    • No space between two selectors.
    • AND relationship between selectors. It chooses the selector with all chained selectors where as grouping is the OR relationship which applies to any selector with one of the mentioned selectors.
    • selector1, selector2 {
           color: red;
      }
    • color:red will apply to any elements that both have selector1 and selector2
    • This syntax works for any combo of selectors except for chaining more than 1 type selectors, because one element cannot have two type selectors at the same time.
  3. Descendant Combinator
    • There is a single space between parent and child selectors.
    • Parent selector precedes child selector.
    • .one .two .three .four
    • The above example selects the class four that's nested within class three. The class three that's nested within class two. The class two that's nested within class one.

Properties to Get Started With

  1. Color and background-color
    1. p {color: #1100ff;} property sets an element's text color
    2. p {color: #1100ff;} sets background color of an element.
    3. several types of values for colors
      1. a keyword: p {color: red; color transparent;}
      2. HEX values: p {color: #1100ff;}
      3. RGB values: p {color: rgb(100, 0, 127);}
      4. HSL values: p {color: hsl(15, 82%, 56%);}
  2. Typography basics and text-align
    1. font family
      1. "Use quotes when font family names contain white spaces"
      2. generic family names never use quotes
      3. It's recommended to list the most desired font first followed by a generic font family as a fallback.
      4. font-family: "Most desired font", generic font family;
    2. font-size
      • The value should never contain any whitespace.
      • font-size: 22px;
    3. font-weight
      1. Boldness of text
      2. Values can be:
        1. A keyword. e.g. font-weight: bold;
        2. A number between 1-1000 (Usually are increments of 100 up to 900) e.g. font-weight: 700;
    4. text-align: center;
      • text-align aligns text horizontally
  3. Image hegiht and width

Adding CSS to HTML

  1. External CSS
    1. embed self closing < link > element inside < head > element.
  2. Internal CSS
    1. embed < style > element inside < head > element
  3. Inline CSS
    1. add a style attribute after the element
    2. inline CSS overrides external and internal CSS, which can cause unexpected results.
    3. not recommended
