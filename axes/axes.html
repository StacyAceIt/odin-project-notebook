Axes

What is the defualt direction of a flex container?

  • Horizontal or row, but you can change it to vertical or column
  • flex-direction sets the main direction. The other direction is the cross direction.

In a column flex-container, what does flex-basis refer to?

when we changed the flex-direction to column, flex-basis refers to height instead of width.

In a row flex-container, what does flex-basis refer to?

width

Why do the above two questions have different answers?

No matter which direction youâ€™re using, you need to think of your flex-containers as having 2 axes: the main axis and the cross axis. It is the direction of these axes that changes when the flex-direction is changed. In most circumstances, flex-direction: row puts the main axis horizontal (left-to-right), and column puts the main axis vertical (top-to-bottom).

Why does the row direction doesn't collapse but column direction collapses?

  • Default Width: The div will expand to fill the width of its containing block.
  • Default Height: The div will have a height of 0 if it is empty.
  • When the direction is set to row, block-level elements default to the full width of their parent div.
  • When changing direction to column, block-level elements default to the height of their content. In this case, there is no content.
