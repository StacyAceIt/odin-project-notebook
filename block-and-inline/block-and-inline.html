What's the difference between block vs inline?

display: block;
  • You can set width and height for block elements.
  • Block elements (display: block) will appear on the page stacked atop each other.
  • Each new element starting on a new line and occupies the full width available.
    •

    display: inline;
  • You can set width or height for inline elements. It occupies as much width as necessary.
  • It doesn't start on a new line.
  • Inline elements appear in line with whatever elements they are place beside.
  • Padding and margin behave differently with inline elements. You don't want to add extra padding or margin on inline elements.

    • display: inline-block;
  • It doesn't start on a new line.
  • You can set height and width for inline-block elements.
  • Behave like inline elements, but with block style padding and margin.
