What's the difference between a flex container and a flex item?

  1. A flex container is any element with display:flex on it.
  2. A flex item is any element that lives directly inside of a flex container.
  3. An element can be both a flex container and a flex item.

How do you create a flex item?

  1. Put display: flex on a flex item.
  2. Then, use flexbox to arrange its children.
