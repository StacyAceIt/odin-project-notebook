Object Basics

Date: Jun 27, 2024

  • What are 2 ways to define an empty object in JS?

    1. const myObject = {};
    2. const myObject = new Object();

  • What does this user[name]; returns?

  • It returns undefined. The correct form is user["name"];

  • What are 2 ways to add/remove/update/access data in an object?

    1. dot notation: obj.single-word-property
    2. square notation: obj["single/multi-word property with quotation marks"]
    3. multi-word properties must be quoted and accessed with square notation.

  • What will be returned here? const person = {}; const a = "a"; person.a;

    • Undefined. To assign a new property with a variable value directly use square bracket instead.

    • person[a]
