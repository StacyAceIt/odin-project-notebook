Object Basics

Date: Jun 27, 2024

  • What are 2 ways to define an empty object in JS?

    1. const myObject = {};
    2. const myObject = new Object();

  • What does this user[name]; returns?

  • It returns undefined. The correct form is user["name"];

  • What are 2 ways to add/remove/update/access data in an object?

    1. dot notation: obj.single-word-property
    2. square notation: obj["single/multi-word property with quotation marks"]
    3. multi-word properties must be quoted and accessed with square notation.

  • What will be returned here? const person = {}; const a = "a"; person.a;

    • Undefined. To assign a new property with a variable value directly use square bracket instead.

    • person[a]

  • What is the other way to write this method? const person = {introduceSelf: function(){}, }

  • const person = {introduceSelf(){}, }

  • What are 2 ways create an object?

    1. create a function that returns an object
    2. use a constructor

  • What is a constructor?

    • A constructor is a function called using the new keyword.

  • Array Methods

    • How to delete an element from an array?

      arr = [0, 1, 2];
    1. delete arr[1]; //returns true and leaves an empty spot in the array
    2. arr.splice(startIdx, deleteCnt, insertVal1, insertVal2, ...)

    What does arr.splice(startIdx, deleteCnt, insertVal1, insertVal2, ...) return?

    Returns the array of removed elements.

    Does arr.splice(startIdx, deleteCnt, insertVal1, insertVal2, ...) allow negative indexes?

    Yes.

    What does arr.slice(startIdx, exclusiveEndIdx) do?

    It returns a new array copying all the elements from startIdx until exclusiveEndIdx

    Does arr.slice(startIdx, exclusiveEndIdx) allow negative indexes?

    Yes.

    What does arr.slice() do?

    it creates a copy of arr array.

    What kind of arguments does arr.concat(arg1, arg2) allow?

    1. arrays: arr.concat([3,4])
    2. values: arr.concat([3,4], 5,6)

    s = "abc" what does s.split() return in JS vs Python?

    1. the returned values are the same in JS and Python
    2. in JS, ['abc']
    3. in Python, ['abc']

    s = "abc" what does s.split("") return in JS vs Python?

    1. split by empty string turns string into characters in JS but returns error in Python.
    2. in JS, ['a', 'b', 'c']
    3. in Python, ERROR

    If no initialValue is given to arr.reduct(function), what is the initialValue by default?

    The first value in the arr.

    What are 3 options when we need to iterate over an array?

    1. forEach
    2. for
    3. for ...of

    What do we use when we need to iterate and return the data for each element?

  • map

    • What will happen if we apply reduce method without initial value to an empty array? let arr = []; arr.reduce((sum, current) => sum + current);

    Error. It's advised to always specify the initial value.
