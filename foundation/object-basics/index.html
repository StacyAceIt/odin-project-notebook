    What are ways to convert a string to a number in JS?

    1. Using the Unary Plus (+) Operator: let a = "1"; +a; // 1
    2. Using the number constructor: let str = "123"; let num = Number(str);
    3. Using parseInt(str, 10) for integers: let str = "123"; let num = parseInt(str, 10);
    4. Using parseFloat(str) for floating point numbers. let str = "123.45"; let num = parseFloat(str);
    5. Using Math.floor(str), Math.ceil(str),Math.round(str) : let str = "123.45"; let numFloor = Math.floor(str); // 123 let numCeil = Math.ceil(str); // 124 let numRound = Math.round(str); // 123

    Why do we need parenthesis to the right side of the arrow function?

    let usersMapped = users.map(user => ({ fullName: `${user.name} ${user.surname}`, id: user.id }));

    This code outputs an array of object. JS identifies left curly bracket as the beginning of a function, not the start of an object. The workout around is to wrap them in the parenthesis.
