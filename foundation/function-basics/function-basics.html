Function Basics

Functions vs Methods

  1. Objects have magic methods (mnemonics).
  2. Methods are the functions of objects.

Parameters vs Arguments

  1. Parameters are placeholders for the arguments that will be passed in when we declare a function.
  2. Arguments are the actual values being passed in when we call a function.

What does function return if it doesn't return a value?

It returns undefined.

What are some well-known function prefixes?

  1. create...
  2. show...
  3. get...
  4. check...

How can we turn a normal function to an arrow function?

  1. remove function keyword
  2. add an arrow after parenthesis for parameters
  3. curly brackets can be omitted if there is only one line inside function body
