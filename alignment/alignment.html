alignment

What's the difference between justify-content and align-items?

  • justify-content aligns items via main axis.
  • align-items aligns items via cross axis.

without flex: 1; default for display: flex is 0 1 auto

with flex: 1, each item grow to fill the available space.

same width with space between.

  • Set justify-content: space-between; to the container.
  • justify-content aligns items across the main axis, which is row by default.

space-around

Set justify-content: center; to container to center boxes along main axis (row).

How to change placement of items along the cross axis (column by default)?

  • Set align-items to container
