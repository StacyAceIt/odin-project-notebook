6 tips to make your code look nicer:

  1. set tab to 2 spaces and indenting should be set to use spaces.
  2. # begins a comment.
  3. method_name, variable_name, file_name always use snake_case.
  4. USE UPPER LETTERS TO DEFINE CONSTANT VALUES
  5. use PascalCase to declare a ClassName
  6. When working with do/end blocks, prefer {curly brackets } when the entire code expression fits 1 line.

What are 2 meanings of API?

  1. API can refer to documentations or docs
  2. It also can refer to Application Programming Interface.
